# 🎯 TINKLEPAD

TINKLEPAD is a lightweight, responsive quiz application built using **HTML, CSS, and JavaScript**. The platform allows users to attempt quizzes based on selected **categories** and **difficulty levels**, with questions being fetched dynamically from a local API source.

Designed as a solo project, TINKLEPAD focuses on providing a clean and interactive quiz-taking experience while demonstrating core front-end development concepts such as DOM manipulation, event handling, API integration, and dynamic UI rendering.

---

## Features

* Dynamic quiz generation using a local API
* Category-based question filtering
* Difficulty-based question selection
* Responsive user interface
* Real-time score calculation
* Interactive quiz navigation
* Result summary upon quiz completion
* Built entirely using Vanilla JavaScript (No frameworks)

## How It Works

1. User selects a quiz category.
2. User chooses the desired difficulty level.
3. Questions are fetched from a local API/data source.
4. The quiz is generated dynamically.
5. User submits answers.
6. The application evaluates responses and displays the final score.

## Planned Features (Future Updates)

The following features are currently under development and will be added in future releases:

### 1. Functional Navbar UI Buttons

The buttons currently present in the navigation bar are part of the UI design and are not yet functional. Their respective functionalities will be implemented in future updates.

### 2. Global Quiz Timer

Currently, timing mechanisms are limited. A dedicated timer for the entire quiz session is planned instead of maintaining timers for individual question slides.

### 3. Enhanced Result Board

The result board currently displays the final score only. Future versions will include:

* User-selected answers
* Correct answers
* Question-wise performance analysis

### 4. Multiple Choice Questions (MCQs)

Support for multiple-answer questions has not been implemented yet and is planned for future releases.

### 5. Question Navigator

A dedicated question navigator panel will be added, allowing users to:

* Jump directly to specific questions
* Track answered/unanswered questions
* Improve overall quiz navigation
